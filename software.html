Software

I've contributed to several open source projects related to the processing and management of spatial data. Most of my developments are written in R, C++, Java, or Python. Some of these projects are listed below. Please see my GitHub profile for more information.

gdalcubes
Earth observation data cubes from satellite image collections (R package and C++ library)
STpconv
Data-driven gap filling of satellite image time series using neural networks with spatiotemporal partial convolutions
bfast
R package for detecting abrupt changes in satellite image time series using "Breaks For Additive Season and Trend"
stmra
R package for the Geostatistical analysis of large environmental datasets with spatiotemporal multi-resolution approximations
